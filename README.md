## Features

- Create tasks instantly
- Mark tasks as complete
- View all tasks per date
- Edit tasks inline by clicking the text
- Delete tasks with confirmation
- Search tasks across all dates
- Organize tasks by date automatically
- Drag and drop to reorder tasks

**Backend:**
- Laravel 12 with Laravel Sail
- Laravel Sanctum authentication
- Repository Pattern
- Resourceful API endpoints
- Form Request validation
- JSON Resources
- Policy-based authorization
- Pest PHP tests
- PSR-12 code style compliance
- MySQL database

**Frontend:**
- Nuxt 4 with TypeScript
- Tailwind CSS styling
- PNPM package manager
- Lucide icons
- Pinia state management
- Modular component structure
